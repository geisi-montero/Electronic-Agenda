# AgendaElectronica

Aplicación de escritorio desarrollada en C# con Windows Forms y Oracle para la gestión de contactos mediante operaciones CRUD.

---

## Tecnologías

- C#
- Windows Forms
- Oracle Database
- Oracle Managed Data Access

---

## Funcionalidades

- Registrar contactos
- Consultar registros
- Modificar información
- Eliminar contactos
- Buscar contactos por nombre
